# Contribuir

## Flujo sugerido

1. crea una rama nueva
2. realiza cambios pequeños y claros
3. documenta lo que modificaste
4. abre un pull request

## Reglas básicas

- no subas videos descargados al repositorio
- no subas cookies reales ni credenciales
- mantén los cambios en PowerShell simples y legibles
- prueba el script antes de proponer cambios
